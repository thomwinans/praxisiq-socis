## Issues Found and Fixed

### 1. LinkedIn Docker container not running
- **Symptom**: `snapp-linkedin` defined in docker-compose but not started
- **Root cause**: Container was never built/started (likely missed during prior sprint)
- **Fix**: `docker compose up -d snapp-linkedin` — built image and started container

### 2. Stale `dotnet test` processes from previous Vera runs
- **Symptom**: Integration tests timing out (504 GatewayTimeout, connection timeouts) despite healthy containers
- **Root cause**: 2 orphaned `dotnet test` processes from prior runs were still hammering Docker services, causing resource exhaustion
- **Fix**: Killed stale processes, restarted Docker containers

### 3. DynamoDB table creation TOCTOU race condition (10 files)
- **Symptom**: `ResourceInUseException: Cannot create preexisting table` — tests randomly failing when run in parallel
- **Root cause**: Every `Ensure*TableAsync()` helper had a check-then-create race: `DescribeTableAsync` throws `ResourceNotFoundException`, then `CreateTableAsync` is called, but another parallel test creates the table between the check and the create, causing `ResourceInUseException`
- **Fix**: Wrapped all `CreateTableAsync` calls in `try { ... } catch (ResourceInUseException) { }` across 10 test files:
  - `test/Snapp.TestHelpers/DynamoDbTestHelper.cs` — `EnsureUsersTableAsync`
  - `test/Snapp.Service.Network.Tests/NetworkIntegrationTests.cs` — `EnsureNetworksTableAsync`
  - `test/Snapp.Service.Notification.Tests/NotificationIntegrationTests.cs` — `EnsureNotificationsTableAsync`
  - `test/Snapp.Service.DigestJob.Tests/DigestJobIntegrationTests.cs` — `EnsureNotificationsTableAsync`
  - `test/Snapp.Service.Content.Tests/ContentIntegrationTests.cs` — `EnsureNetworksTableAsync`, `EnsureContentTableAsync`, `EnsureNotificationsTableAsync`
  - `test/Snapp.Service.Transaction.Tests/DealRoomIntegrationTests.cs` — `EnsureTxTableAsync`
  - `test/Snapp.Service.Transaction.Tests/TransactionIntegrationTests.cs` — `EnsureTxTableAsync`, `EnsureNetworksTableAsync`
  - `test/Snapp.Service.Intelligence.Tests/IntelligenceIntegrationTests.cs` — `EnsureNotificationsTableAsync`, `EnsureIntelTableAsync`
  - `test/Snapp.Service.Intelligence.Tests/UnlockIntegrationTests.cs` — `EnsureTransactionsTableAsync`, `EnsureIntelTableAsync`
  - `test/Snapp.Service.Intelligence.Tests/QuestionIntegrationTests.cs` — `EnsureIntelTableAsync`
  - `test/Snapp.Service.Intelligence.Tests/CompensationIntegrationTests.cs` — `EnsureIntelTableAsync`
